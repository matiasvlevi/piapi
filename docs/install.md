### Server Fetch | Documentation

## Install

Clone the repo

```
git clone https://github.com/matiasvlevi/serverfetch.git
```

Install dependencies, and required configuration files
This will create a plain `.apirc` and `.env` file.

```sh
npm run config-win 
# or
npm run config-linux
```
